
# 简介

K6：负载测试脚本编写、性能测试执行、分布式测试、实时监控等。

与其他主流压测工具相比，K6 具有以下优势：

1. **现代化架构**：K6 基于 Go 语言开发，具有轻量级、高性能的特点。
2. **简单易用**：K6 提供了简洁的脚本编写语法，易于上手和使用。
3. **实时监控**：K6 可以实时监控测试执行过程中的性能指标，并生成详细的报告。
4. **开源免费**：K6 是开源工具，免费提供给用户使用。
